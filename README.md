# williamjharris.github.io

Personal portfolio site for Liam Harris — Biomedical Engineering student at CU Boulder.

Built with plain HTML, CSS, and vanilla JavaScript. Hosted on GitHub Pages.

## Features

- Dark techy aesthetic with neon green accents
- Terminal-style hero intro with typing animation
- 3D CSS bookshelf for project showcase
- Animated social links with icon popups on hover
- Fully responsive, no build tools or frameworks
