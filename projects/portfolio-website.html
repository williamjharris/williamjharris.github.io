Personal Portfolio Website

A portfolio site built from scratch with plain HTML, CSS, and vanilla JavaScript. No frameworks, no dependencies — just the fundamentals.

2025 — Present
HTML CSS JavaScript GitHub Pages
View on GitHub →

Overview

This is the site you're looking at right now. I built it to showcase my projects and skills as a biomedical engineering student, and as a learning exercise in front-end web development. Everything is hand-written — no React, no Tailwind, no build tools.

Features

  • Terminal-style typing animation on the landing page
  • 3D CSS bookshelf for project display with click-to-open interaction
  • WebGL animated background (topographic neon grid)
  • Animated social links with icon popups
  • Fully responsive design for mobile and desktop
  • Hosted on GitHub Pages with custom domain

Technical Details

The WebGL background uses a custom fragment shader that generates a procedural topographic grid effect with noise-based terrain. The bookshelf is built entirely with CSS 3D transforms — each book is a six-sided box that rotates open on click, revealing a details panel.

What I Learned

Building a site without frameworks forced me to deeply understand CSS layout, 3D transforms, and vanilla JavaScript DOM manipulation. Writing the WebGL shader from scratch was a great introduction to GPU programming and GLSL.
