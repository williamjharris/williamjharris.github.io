Wearable Heart Rate Monitor

Real-time BPM monitoring with an ESP32 microcontroller and MAX30102 pulse oximeter sensor, featuring a WiFi web dashboard and Python data logger.

2025 — Present
ESP32 MAX30102 C++ Python WiFi
View on GitHub →

Overview

This project is a wearable heart rate monitor built from scratch using an ESP32 microcontroller and MAX30102 pulse oximeter sensor. It reads real-time BPM data and displays it on a 0.96" OLED screen, while simultaneously serving a live web dashboard over WiFi and logging data to a Python script for long-term analysis.

Hardware

  • ESP32 DevKit V1 — WiFi-enabled microcontroller
  • MAX30102 — Pulse oximeter and heart rate sensor (I2C)
  • SSD1306 0.96" OLED — Real-time BPM display
  • 3.7V LiPo battery with TP4056 charging module

Software Architecture

The firmware is written in C++ using the Arduino framework. It reads raw IR data from the MAX30102, applies a peak-detection algorithm to calculate BPM, and pushes updates to three outputs simultaneously: the OLED display, a WebSocket-based web dashboard, and a serial data stream consumed by the Python logger.

Web Dashboard

The ESP32 hosts a lightweight web server that serves an HTML/JS dashboard over WiFi. The dashboard connects via WebSocket for real-time BPM updates and renders a live-updating chart of heart rate over time.

Python Data Logger

A companion Python script connects to the ESP32's serial output and logs timestamped BPM readings to a CSV file. This enables long-term heart rate trend analysis using pandas and matplotlib.
